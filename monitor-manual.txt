.SH NAME
Monitor \-is a tool to monitor a directory for any changes. system 
.SH SYNOPSIS
./monitor
.SH DESCRIPTION
Linux does not provide a tool to monitor a directory for any changes, so I created a script which prints out any changes that happen in that directory.
